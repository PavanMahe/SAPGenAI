####1. Setup the cloud DaaS from aiven.io for Postgres
- Install a tool called pgAdmin (a tool to connect to DB) (like it as HANA studio)
https://www.pgadmin.org/download/pgadmin-4-windows/
- Setup free account with aiven.io
- Get your DB connected to your pgAdmin
- Create a .env file in your project directory and maintain the details as below
DB_URL="postgres://.......YOUR CONNECTION STRING"
